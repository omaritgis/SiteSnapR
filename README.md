# SiteSnapR
## Python site snapper - Shady link? Try it in sitesnapr

## Setup

Installation:
```
git clone https://github.com/omaritgis/SiteSnapR.git
docker build -t sitesnapr .
docker run -it sitesnapr
```

Usage:
Select protocol (http/https), enter url and submit!
